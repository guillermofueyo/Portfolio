# Proyectos

## App de Recetas Kotlin MVVM

Aplicación móvil orientada a la gestión y visualización de recetas culinarias, implementando arquitectura MVVM, navegación entre pantallas y sistema de login.

### Tecnologías utilizadas

* Kotlin
* Android Studio
* MVVM
* Navigation Component

### Repositorio

https://github.com/guillermofueyo/App_Movil_Recetas.git


## App NUI-VUI Control de Brillo por Voz

Aplicación Android desarrollada en Java basada en interacción NUI/VUI para controlar el brillo del dispositivo mediante comandos de voz.

### Tecnologías utilizadas

* Java
* Android SDK
* RecognizerIntent
* ConstraintLayout

### Repositorio

https://github.com/guillermofueyo/App_NUI_VUI.git


## Gestión de Empresa Circense

Aplicación de escritorio desarrollada en Java 23 con Spring Boot y JavaFX para la gestión integral de una empresa circense, integrando múltiples sistemas de persistencia.

### Tecnologías utilizadas

* Java 23
* Spring Boot
* JavaFX
* MySQL
* MongoDB
* DB4O
* ObjectDB
* eXistDB
* Maven

### Repositorio

https://github.com/guillermofueyo/Gestion_de_Empresa_Circense_Aplicativo_Java_Full_Stack.git


## Proyecto Integrador IA y Análisis de Datos

Proyecto desarrollado en Google Colab utilizando Excel, Power BI y Python para análisis, automatización y visualización de datos.

### Tecnologías utilizadas

* Python
* Google Colab
* Excel
* Power BI
* Pandas
* NumPy
* Matplotlib

### Funcionalidades

* Procesamiento y limpieza de datos
* Automatización de tareas
* Análisis estadístico
* Visualización de datos
* Predicción básica mediante IA

### Archivo principal

* `ProyectoFinal.ipynb`

### Repositorio

https://github.com/guillermofueyo/Proyecto_Integrador_IA_Analisis_Datos.git
